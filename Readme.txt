Pasos para realizar:
1- Hacer un clone de git. https://github.com/paloga/ExamenVagrant.git
2- Crear una máquina con el init.
3- Metiendo dentro de la carpeta que se genera el Vagrantfile, el provision.sh y la carpeta freelancer.
4- Despues hacemos un "vagrant up".
5- Una vez echo esto ejecutaria el provision.sh con el comando: vagrant provision.
6- Despues con el comando "ssh -p 2222 vagrant@localhost" entrariamos a la máquina la contraseña es: vagrant.
7- Accediendo a http://localhost:8181/ accederiamos a la web a programar. Y en la carpeta: C:\cygwin64\home\usuario\maquina\freelancer-16-17 se encontraria el index.
